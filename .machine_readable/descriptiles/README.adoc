# A2ML 6a2 Directory — IDApTIK

This directory contains the 6 core A2ML machine-readable metadata files for
IDApTIK.

## Files

- `AGENTIC.a2ml` - AI agent operational gating, safety controls, licence invariants
- `ECOSYSTEM.a2ml` - Project ecosystem position, relationships, explicit boundaries
- `META.a2ml` - Architecture decisions (ADR-0001..0006), development practices, design rationale
- `NEUROSYM.a2ml` - Symbolic rules, scan configuration, config-layer semantics
- `PLAYBOOK.a2ml` - Operational runbooks (just recipes for build/run/verify)
- `STATE.a2ml` - Project state, roadmap phase, component status, route to MVP

## Standards Compliance

These files follow the A2ML Format Family specification from:
https://github.com/hyperpolymath/standards/tree/main/a2ml

Validate this set with the a2ml-validate-action script (`validate-a2ml.sh`).
